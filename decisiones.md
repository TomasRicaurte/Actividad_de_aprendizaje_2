**Sitio web elegido** 

Magic Baby

e-commerce enfocado en productos para bebes e infantes

**Roles para los usuarios:**

1. Joven con problemas motrices (persona 1)

2. Madre invidente que usa tecnologias asistivas (persona 2)

3. Adulto mayor con problemas visuales y equipo lento (persona 3)

**Anecdotas de las personas**

**Persona 1**

Anecdota 1: Al ser una persona con problemas motrices me es indispensable el poder navegar por una pagina web unicamente con mi teclado y lograr realizar todas las acciones necesarias para comprar sin inconvenientes mayores.

Criterios WCAG vistos:
WCAG 2.1.1 Keyboard /
WCAG 2.4.7 Focus Visible /
WCAG 2.4.1 Bypass Blocks 

Anecdota 2: De vez en cuando con el uso de mi celular tambien se me dificulta navegar si un sitio web no cuenta con botones y enlaces  los suficientemente grandes para no equivocarme al seleccionar las opciones.

Criterios WCAG vistos:
WCAG 2.5.8 Target Size - Nivel AA en WCAG 2.2 /
WCAG 2.5.7 Dragging Movements


**Persona 2**

Anecdota 1: Al ser una madre invidente que usa tecnologias asistivas es crucial que los sitios web tengan definidos sus campos para que se me pueda describir que hay en la pantalla y realizar mis compras sin que sea una odisea.

Criterios WCAG vistos:
WCAG 1.1.1 Non-text Content /
WCAG 4.1.2 Name, Role, Value /
WCAG 1.3.1 Info and Relationships 

Anecdota 2:Uno de los inconvenientes que me ha pasado es cuando un formulario o dato requerido esta incorrecto o no lo leyo correctamente mi lector de pantallas, no se me indica exactamente que es lo que hay que corregir y no puedo continuar sin ayuda de alguien más lo que es frustrante.

Criterios WCAG vistos:
WCAG 3.3.1 Error Identification /
WCAG 3.3.3 Error Suggestion /
WCAG 4.1.3 Status Messages


**Persona 3**

Anecdota 1: Yo ya soy un adulto mayor, mi vision no esta bien y se me dificulta la lectura de la letra en mi telefono o computadora, necesito poder agrandar la pantalla para ver que es lo que hay en el sitio web.

Criterios WCAG vistos:
WCAG 1.4.4 Resize Text /
WCAG 1.4.3 Contrast (Minimum) /
WCAG 1.4.10 Reflow

Anecdota 2: Tampoco tengo un telefono muy bueno y la computadora es de las baratas, muchas veces los sitios se tardan mucho en abrir y en ese tiempo me distraigo o se me olvido que es lo que necesitaba hacer.

Criterios WCAG vistos:
WCAG 2.2.2 Pause, Stop, Hide /
WCAG 1.3.2 Meaningful Sequence /
Relacionado con robustez y experiencia de usuario accesible
